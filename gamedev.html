GAME DEVELOPMENT PROJECT

The theme of this project was 'escape'. Using this theme I came up with the idea that my game would be a horror game set in an Asylum. The player would have to find and collect all the keys in order to unlock doors and escape the Asylum. I created this game using unreal engine and created all the 3D models using 3Ds Max.
