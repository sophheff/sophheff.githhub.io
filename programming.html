ENCRYPTION PROJECT

Encryption is where plaintext is encoded to appear as random characters, usually the encrypted message can be decrypted using an encryption key. The aim of my project was to create a program using python that could do the basics of encryption. The program will begin by presenting you with a 4 option menu which allows you to generate an encryption key, encrypt a message, decrypt a message and close the program.
